Installatie

De eenvoudigste manier om ttb te installeren is via een kant-en-klaar pakket. Bijvoorbeeld: Debian en Ubuntu kunnen ttb gewoon installeren via het installatieprogramma. Voor Ubuntu, zie het screenshot.

Als er geen kant-en-klaar pakket beschikbaar is, kan ttb ook redelijk eenvoudig handmatig ge√Ønstalleerd worden, met de volgende stappen:

  • Download ttb-1.0.tar.gz
  • Vanaf de commandline: 
    
      $ tar xvfz ttb-1.0.tar.gz
      $ cd ttb-1.0/
    En nu als rootgebruiker (of 'sudo'): 
    
      # python ./setup.py -v install
  • Als alles goed is gegaan staat ttb nu in het "startmenu" van GNOME/KDE/...; of ttb kan gestart worden vanaf de commandline, met het command ttb (of zelfs ttb 101 om direct naar het laatste nieuws te gaan).
